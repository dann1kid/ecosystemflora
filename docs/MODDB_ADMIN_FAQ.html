Slower spread (global)
Set preset to sparse — fewer attempts, stricter fitness, wider spacing.
"BalancePreset": "sparse"
Manual tuning? Use "custom" then edit ReproduceAttemptsPerYear, ReproduceChance, MinFitness. Presets natural/lush/sparse overwrite those keys on every server start.
Treehouses / bases
Only wild log-grown trunks are in ecology. Player log-placed / debarked builds are not aged or killed by the mod.
Living inside natural worldgen trees? Those trunks can senesce. Use land claims — spread, growth, and senescence are blocked inside claims.
No wild tree death
Growth off and senescence off:
"EnableTreeAging": false,
"EnableTreeSenescence": false
Want slow growth but no death? Aging on, senescence off. There is no config key for lifespan yet — horizons are per species in code (redwood ~140 y).
Do not run alongside Wild Farming Revival. Vintage Story 1.22+. Press I on a trunk log for calendar age, size index, and senescence phase.
Frequently asked
Is redwood affected? Yes — one of 14 lumber species (log-grown-redwood). Sapling spread, optional yearly growth, senescence at ~140 in-game years (longest horizon). Redwood is conifer-like — no deciduous autumn leaf drop from seasonal canopy.
Increase tree lifespan? No per-species lifespan slider in config yet. Disable EnableTreeSenescence to prevent natural death. Species horizons (oak ~120, birch ~90, redwood ~140) are fixed in mod data.
Slow everything down? BalancePreset: sparse is the main global knob. Finer control: custom + lower ReproduceAttemptsPerYear (default 72) and ReproduceChance (default 0.5). Per-species SpreadRate in ecology tables still applies on top.
Meadow too dense? sparse preset, or custom with higher MinFitness and spacing. Gallery screenshots may use lush/aggressive tuning — default is natural.
Safe gardens? Land claims block wild spread, displacement, stress death, tree growth, and senescence. Greenhouse (enclosed glass roof) protects from temperature stress without claims.
